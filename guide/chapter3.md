## 起点

在消耗了 100μg 后，我们可以解锁五条皮亚诺公理 apn1–apn5 和 10 以内的数字定义

```txt
apn1:(0@N)
0 是自然数

apn2:(Vx:((x@N)>(S(x)@N)))
自然数的后继是自然数

apn3:(Vx:~(S(x)=0))
对于任意数 x，x 的后继数不为零（保证 0 是第一个自然数）

apn4:(Vx:(Vy:((S(x)=S(y))>(x=y))))
后继数相同的数相同（避免分支）

apn5:(#rp(#nf($0,x),$1,0)>((Vx:((x@N)>(#rp(#nf($0,x),$1,x)>#rp(#nf($0,x),$1,S(x)))))>(Vx:((x@N)>#rp(#nf($0,x),$1,x)))))
数学归纳法
```

## 侧翼

### ~(1=0)

###### 引理 s0

- ($0=$1)⊢($1=$0)

```deduct
hyp $0=$1
a8 $0 $1 $0=$0 1
mp 1 0
a7 $0
mp 2 3
```

##### 证明

```deduct
apn3
a4 x ~(S(x)=0) 0
mp 1 0
a8 S(0) 1 ~(S(0)=0) 0
d1
s0 4
mp 3 5
mp 6 2
```
